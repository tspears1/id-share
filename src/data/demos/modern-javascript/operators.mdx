## Comparison Operators

| Operator | Description |
| -------- | ----------- |
| `==`     | Equal to |
| `!=`     | Not equal to |
| `===`    | Strict equal to |
| `!==`    | Strict not equal to |

## Logical Operators

| Operator | Description |
| -------- | ----------- |
| `&&`     | Logical AND |
| `||`     | Logical OR |
| `!`      | Logical NOT |

## Nullish Coalescing Operator

The nullish coalescing operator (`??`) returns the value of the first expression if it is not `null` or `undefined`, otherwise returns the value of the second expression.
